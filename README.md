# ikunPlatformGame

#### 介绍
这是一个个人自作的平台小游戏，主角是一个会唱跳rap和篮球多才多艺的小鸡

####展示
![输入图片说明](https://foruda.gitee.com/images/1704980699151904465/e1be5aba_13879088.png "ikun_jump.png")

#### 使用说明

1.  克隆/下载 zip文件
2.  解压文件
3.  用unity打开即可查看
